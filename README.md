# 🚀 Arduino Bluetooth Control System

Yeh ek hardware-software integration project hai jo smartphone ke zariye devices ko wirelessly control karne ke liye banaya gaya hai.

### 🛠 Hardware Required:
* **Arduino Uno/Nano**
* **HC-05 or HC-06 Bluetooth Module**
* **Jumper Wires**
* **Smartphone with Bluetooth Control App**

### 💻 Software & Language:
* **Language:** C++ (Arduino Sketch)
* **Tool:** Arduino IDE

### ⚙️ How it Works:
1. Bluetooth module ko Arduino se connect karein (TX to RX, RX to TX).
2. Code (.ino file) ko Arduino IDE ke zariye upload karein.
3. Mobile app se module ko pair karein.
4. App se commands bhejein (jaise '1' for ON, '0' for OFF).
